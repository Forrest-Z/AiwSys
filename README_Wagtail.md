# Wagtail
    Wagtail 是一个Django框架的开源的CMS框架
    ```
    pip install django  --user    
    pip install wagtail  --user  
    #启动建立一个工程    
    wagtail start mysite[你的工程名]    
    #进入工程目录   
    cd mysite  
    pip install -r requirements.txt  
    ./manage.py migrate    
    ./manage.py createsuperuser    
    ./manage.py runserver
    ```
